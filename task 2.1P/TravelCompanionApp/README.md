# Travel Companion Converter App (SIT305 - Task 2.1P)

## 📱 Overview
This is an Android mobile application developed for SIT305 Task 2.1P.

The app is a Travel Companion Converter that allows users to convert values between different units, such as:
- Temperature (Celsius, Kelvin)
- Currency (USD, AUD)

## 🚀 Features
- Simple and clean user interface
- Unit conversion functionality
- Supports multiple categories (Temperature, Currency)
- Instant calculation and result display

## 🛠️ Technologies Used
- Java
- Android Studio
- XML (UI Design)

## 📂 Project Structure
- `MainActivity.java` – Handles UI and conversion logic
- `activity_main.xml` – Layout design
- Conversion logic implemented directly in the app
